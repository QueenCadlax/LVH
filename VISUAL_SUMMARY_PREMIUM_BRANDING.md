## The 8 Strategic Principles

```

│           THE 8 CORE STRATEGIC PRINCIPLES               │
├─────────────────────────────────────────────────────────┤
│                                                         │

│      └─ We select. You get quality.                    │

│      └─ Repeat themes. Build muscle memory.           │
│                                                         │

│      └─ "Hospitality" not "Stays"                     │
│                                                         │
│  7️⃣  SUBSTANCE, NOT REPETITION                        │
│      └─ Each message adds unique value.               │
```
