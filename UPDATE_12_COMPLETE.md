## Code Changes

**File:** `App.tsx` (lines ~4609-4620)

### What Changed

```diff
- className="text-[0.95rem] uppercase tracking-[1.5px] font-medium"
+ className="text-[1rem] tracking-[0.5px] font-medium"
```

### Why

| Change | Reason |
|--------|--------|
| Removed `uppercase` | Sentence case feels more premium |
| `0.95rem` → `1rem` | Better readability at normal case |
| `tracking-[1.5px]` → `tracking-[0.5px]` | Less spread-out, more elegant |
