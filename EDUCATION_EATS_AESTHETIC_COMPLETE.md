## 🔧 Technical Details

### Color System (CSS Classes Used)

**Background Colors:**
```css
bg-black          /* Main background */
bg-black/60       /* Cards, semi-transparent */
bg-black/70       /* Search bar backdrop */
bg-black/80       /* Darker elements */
bg-black/95       /* Modal background */
```

**Text Colors:**
```css
text-white        /* Primary text */
text-gray-300     /* Secondary text, headings */
text-gray-400     /* Tertiary text, descriptions */
text-gray-200     /* Input text */
text-yellow-400   /* Gold accents, stars */
text-yellow-300   /* Hover state for gold */
text-black        /* On gold backgrounds */
```

**Borders:**
```css
border-white/10   /* Subtle card borders */
border-yellow-400/50   /* Hover state */
```

**Shadows:**
```css
shadow-lg hover:shadow-yellow-400/20   /* Gold glow on hover */
```
