## 🎬 USER INTERACTION FLOWS

### **Search Flow**
```
User Types "Smith"
        ↓
Input Triggers onChange
        ↓
setSearchTerm("Smith")
        ↓
Component Re-renders
        ↓
Filter Logic Runs:
- Searches name & specialty
- Matches: "Smith", any specialty
- Also applies other active filters
        ↓
Display Updated Results
Results: Dr. John Smith (General Practitioner)
```

### **Filter Flow**
```
User Selects "Cardiologist"
        ↓
setSelectedSpecialty("cardiologist")
        ↓
filteredProviders recalculates with:
- Search term (if any)
- Specialty = "cardiologist"
- Location filter (if any)
- Verified toggle (if on)
        ↓
Display Only Cardiologists
Matching All Active Filters
```

### **Quick Button Flow**
```
User Clicks "Dentists"
        ↓
Button onClick Handler Fires
        ↓
setSelectedSpecialty("dentists")
        ↓
Refilter All Providers
        ↓
Show Only Dentist Providers
(Instantly updates "All Providers" listing)
```
