
## 📐 Responsive Grid

### Mobile (320px - 640px)
- Single column layout
- Full-width inputs
- Stack all sections vertically
- Single column for package cards

### Tablet (640px - 1024px)
- 2-column grid for some sections
- Narrower padding
- 2-column package cards

### Desktop (1024px+)
- Optimal spacing
- 2-column for form sections
- 3-column for package cards
- 2x2 grid for Step 5 summary
