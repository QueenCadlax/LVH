## 📐 Typography Hierarchy (AFTER REDESIGN)

```
MAXIMUM SIZES:
┌─────────────────────────────────────────────────┐
│  Hero Title:     5xl = 48px max (was 6xl/60px) │ ← KEY: No oversized
│  Section H2:     4xl = 36px                     │    fonts anymore
│  Card H3:        xl  = 20px                     │
│  Body Text:      sm  = 14px                     │
│  Fine Print:     xs  = 12px                     │
└─────────────────────────────────────────────────┘

SIZE REDUCTIONS:
├─ Hero:         60px → 48px  (-20%)  Elegant refinement
├─ Sections:     48px → 36px  (-25%)  Premium feel
├─ Features:     14px → 12px  (-14%)  Still readable
├─ Padding:      80px → 64px  (-20%)  Less sprawl
└─ Icons:        16px → 14px  (-12%)  Proportional
```

├─ Usage:       Platinum package card

├─ White:       Headings, primary text
